★java의 연산자<br>

<, >, <=, >=, ==(같다면)<br>
!= : 다르다면<br>
++ --(증감)<br>
a++ : 나중에증가 시켜라(중요함)<br>
++a : 먼저 증가 시켜라(중요함)<br>
&&(and) , ||(or)<br>
