* 튜플은 리스트와는 다르게 값을 자유롭게 바꿀 수 없고, []대신 ()안에 넣거나 괄호자체를 생략하고 만든다.

{% highlight python %}
>>> t1 = ()
>>> t2 = (1,)   //  요소가 하나인 경우 콤마찍어줘야 함
>>> t3 = (1, 2, 3)
>>> t4 = 1, 2, 3
>>> t5 = ('a', 'b', ('ab', 'cd'))
{% endhighlight %}

* 값을 바꿀 수 없기 때문에 값이 바뀌면 안되는 상황에서 사용하면 된다.

* 인덱싱, 슬라이싱은 리스트와 같이 t1[index]로 한다.

* +, * 연산자로 두 개의 튜플을 합치거나 반복할 수 있다.

* len(tuple) : 길이를 구한다.
